Rider developed using intellij
Brianna developed using notepad++ and the command line


How To Run Program:

- On file input have spaces between tokens
- After a line end with a space or newline 


To Compile Through Command Line:

- Ensure javac compiler is included in the PATH environment variable
- Navigate to the directory that contains all the java source files
- Enter "javac *.java" into the command line to compile all the source files
- Enter "java Parser [filename]" to run the Parser on any text file
